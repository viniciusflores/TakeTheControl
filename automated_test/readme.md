### How to test

Clone project:
> git clone <https://github.com/viniciusflores/TakeTheControl.git>

Access project directory
> cd automated_test

Install dependencies
> yarn

Run all Tests
> yarn cy:run

Run all Tests, generate report and
> yarn test

To see the Cypress Dashboard
> yarn cy:open
